# Formulário de Equipamentos

Formulário web para cadastro de equipamentos de TI com integração ao Google Sheets.

## Campos do Formulário

- **Portador**: Nome da pessoa responsável pelo equipamento
- **Equipamento**: Tipo de equipamento (MacBook, Dell, Lenovo, etc.)
- **Modelo**: Versão/modelo do equipamento
- **Serial Number**: Número de série único
- **HD**: Capacidade de armazenamento
- **Processador**: Tipo de processador
- **Memória**: Quantidade de RAM
- **Marca**: Fabricante do equipamento
- **Seu e-mail**: Email de contato
- **Foto como evidência**: Anexo de imagem para comprovação

## Como Usar

1. Acesse o formulário através do link do GitHub Pages
2. Preencha todos os campos obrigatórios
3. Anexe uma foto do equipamento como evidência
4. Clique em "Enviar Formulário"
5. Os dados serão salvos automaticamente na planilha Google Sheets

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Sheets API
- Sheet.best (para integração com Google Sheets)
